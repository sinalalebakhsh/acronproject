Project Overview

acronproject.com is a Django-based web application currently under development. The project focuses on creating a comprehensive user panel for editors to publish articles. Additionally, it aims to develop a messaging system similar to Telegram, where users can send messages to each other. The messaging feature is designed to exclude image and video sharing to minimize server load.


بررسی پروژه

acronproject.com یک برنامه وب مبتنی بر Django است که در حال حاضر در حال توسعه است. این پروژه بر ایجاد یک پنل کاربری جامع برای ویراستاران به منظور انتشار مقالات تمرکز دارد. علاوه بر این، هدف آن توسعه یک سیستم پیام‌رسانی مشابه تلگرام است که کاربران بتوانند به همدیگر پیام دهند. این ویژگی پیام‌رسانی به گونه‌ای طراحی شده که ارسال تصاویر و ویدیوها را حذف کرده تا بار سرور به حداقل برسد.
